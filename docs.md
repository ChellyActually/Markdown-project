# Markdown notepad

## File structure
```
|- index.html
|- main.js    (turns text in input field to string)
|- preview.js     ()
```

## Main.js
```
updatePreview {
    text = get element (field)
    html = mdToHtml(text)
    innerhtml = html
}
```

## preview.js
```
mdToHtml(raw) {
    return html
}
```
